## 线路实测：移动用户直接买，电信用户先看这里

三网的体验分化比较明显，简单说清楚：

**移动用户**：去程、回程全程 CMI 直连，延迟最低能跑到 20-30ms（广东方向），全国平均在 50ms 出头。白天晚上基本稳，是这个套餐真正的目标用户。

**联通用户**：回程走移动 CMI，去程搬瓦工做过优化，改成了联通 AS4837 直连，整体比早期好了很多。晚高峰偶尔有波动，丢包比移动稍高，但总体可以接受。

**电信用户**：早期去程绕 NTT 日本，搬瓦工后来做了一次调整，电信去程优化成了 CN2 GIA，变化挺大的。不过历史上电信在这个机房确实有晚高峰不稳的记录，如果你是重度电信用户且对稳定性要求很高，还是建议慎重。

磁盘读写这块，实测 IO 性能不错，海外方向实际带宽据测可以跑到接近 10Gbps，到国内是走 1Gbps 限速。
